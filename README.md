# Cancer Multi-Omics Survival Pipeline

Reproducible bioinformatics workflows for integrating transcriptomics, proteomics, mutation-derived features, and clinical metadata in cancer survival analysis.

## What this project demonstrates
- Multi-omics data integration
- Survival analysis (Cox proportional hazards)
- Transcriptomics and proteomics workflows
- Clinical metadata harmonisation
- Reproducible analysis in R, shell, Git, and Linux/HPC

## Repository overview
- `01_transcriptomics/` – preprocessing and modelling of transcriptomic features
- `02_proteomics/` – preprocessing and modelling of proteomic features
- `environment.yml` – computational environment
